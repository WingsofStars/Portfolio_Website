Auxilium

What is Axuilium

Auxilium is a Free to use, completely open source Virtual assistant that can run on a raspberry pi 4. The github repo can be found here

How to get Started

To get started with this project, you'll need a Raspbery Pi 4 with a minimal requirment of 2 Gigabytes of Ram, an Adafruit Voice Bonnet for Raspberry Pi and atleast 1 Adafruit mono speaker. After the easy installation of those two parts to the raspberry pi, download the OS image here, and flash it to an micro SD card thats atleast 16 GB. You can use a flasher such as Balena Etcher to install it to the micro SD card.
